# Structural Theory (SDO)

Structural Theory (SDO) starts from a minimal structural triad: relational difference, recursive self-binding, and openness. The project investigates how such relations can generate stable coupling spaces and how these structures become visible as projection geometry, locality, and entanglement-like relations.

## Visual entry point

The most direct entry into the current operational model is the interactive Coupling Graph Explorer:

[Open the Coupling Graph Explorer](https://soz96aufhj70geti-cmyk.github.io/structural-theory-sdo/)

The explorer visualizes admissible recombination relations between path-pair states. Nodes represent pair states in the path space; edges represent admissible coupling operations. The viewer is intended as the primary visual access point to the current operator model.
